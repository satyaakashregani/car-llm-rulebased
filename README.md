# 🚗 Car LLM (Rule-Based Car Assistant)

A Python-based car assistant that answers queries about Indian cars (2000–2026) using a rule-based system — no external APIs.

## 🔧 Features

* Fuzzy search (handles typos & aliases)
* Intent detection (price, engine, features, etc.)
* Car comparison (e.g., "Creta vs Seltos")
* CLI-based interaction

## ⚙️ Tech

* Python
* OOP (Classes & Objects)
* difflib (fuzzy matching)
* Rule-based logic

## ▶️ Run

```bash
python main.py
```
